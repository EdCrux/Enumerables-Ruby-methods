## Getting Started

Install ruby 2.6.5
Clone the repo
Bundle init

### Prerequisites

- Ruby version ~> 2.6
- IDE (if you want to test)

### Dependencies

- Rubocop

### Install

`bundle install`

### Run tests

Rubocop lint

`rubocop enumerables.rb`

### Deployment

`ruby enumerables.rb`
